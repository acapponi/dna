Nav to dropdown

Lista de links que se transforma en un dropdown en el breakpoint elegido. La estructura es identica a la estructura original de los dropdown de Bootstrap, pero cambian las clases del div parent.

La clase sin modificar (nav-to-dropdown) genera un dropdown funcional, mientras que el modificador (nav-to-dropdown-**) indica a partir de que breakpoint ese dropdown se muestra como lista. Se puede usar cualquiera de los breakpoints que usamos normalmente.

La lista mantiene los estilos básicos del dropdown menu, pero se pueden pisar fácilmente.

Dropdown link
Action Another action Something else here 
<div class="nav-to-dropdown nav-to-dropdown-md">
  <a class="btn btn-secondary dropdown-toggle" href="#" role="button" id="dropdownMenuLink" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
    Dropdown link
  </a>

  <div class="dropdown-menu" aria-labelledby="dropdownMenuLink">
    <a class="dropdown-item" href="#">Action</a>
    <a class="dropdown-item" href="#">Another action</a>
    <a class="dropdown-item" href="#">Something else here</a>
  </div>
</div>
