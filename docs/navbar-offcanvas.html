Navbar offcanvas

Navbar con clases especiales que generan, en el breakpoint deseado, que el menu se abra desde la izquierda (offcanvas)

La estructura es igual a la navbar tradicional con el agregado de la clase .offcanvas-collapse en el collapse.

Es importante, como en toda navbar, no olvidar agregar la clase navbar-expand-{breakpoint} indicando desde donde que quiere ver el menu mobile.

  <div class="container-fluid">
    <a class="navbar-brand" href="#">Offcanvas navbar</a>
    <button class="navbar-toggler p-0 border-0" type="button" data-bs-toggle="offcanvas" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>

    <div class="navbar-collapse offcanvas-collapse" id="navbarsExampleDefault">
      <ul class="navbar-nav me-auto mb-2 mb-lg-0">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Dashboard</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Notifications</a>
        </li>

          <a class="nav-link" href="#">Profile</a>
